# Lil Adventurer

Lil Adventurer is a 3D horror-adventure survival game developed in Unity.

## Gameplay
- Explore a dark forest
- Collect relics
- Avoid enemy creatures
- Unlock the escape gate
- Escape before losing all health

## Built With
- Unity Engine
- C#
